This module extends the Odoo Sale Blanket Order module to adapt it to the Brazilian needs, with this module you have tax data for collection and generation of fiscal documents (NF-e, NFS-e, CF-e, NFC-e and others), calculation Brazilian taxes and contributions (municipal, state and federal).

Usage
=====

To use this module, you need to:

#. Go to Sale
#. Create or select a Sale Blanket Order
#. Confirm the Sale Blanket Order
#. Create Sale Order
