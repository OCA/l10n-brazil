This module enables the sending of Service Invoices via the FocusNFE API.

For more information, visit: https://focusnfe.com.br/

Installation
============

To install this feature, simply install the module.

Configuration
=============

To configure this module, simply access the company registration, in the NFSe tab, set up the provider, environment, and token.

#. Settings > Users & Companies > Companies
#. Select the desired Company
#. Navigate to the Fiscal Page
#. Go to the NFS-e Subsection
#. Choose NFSe Provider as FocusNFE

Usage
=====

To use this module:

#. Create an invoice with the fiscal document type 'SE'.
#. Fill in the necessary details such as the city's tax code, taxes, and related information.
#. Validate the document.
#. Send the Fiscal Document.
#. Monitor the document's processing status.
