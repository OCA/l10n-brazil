Installation
============

This module depends on:

* account_payment_order
* account_due_list
* l10n_br_account

Configuration
=============

No configuration required.

Usage
=====

This module provides a menu to configure payment order types : Accounting > Configuration > Miscellaneous > Payment Export Types
